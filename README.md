## About The Project

<div align="center">

Web page app created using raw classic web development languages.

You can create an account to customize your Gaimon (OnePiece character) and save it in a database!

<hr/>

## Built With

- Front-end: HTML, CSS, JS
- Back-end: PHP, MySQL
